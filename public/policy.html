Privacy & Data

Inventory stores your data locally in your browser (per user). If you enable Cloud Sync, your data is mirrored to your own Firebase Realtime Database account under your authenticated user ID.

  • Local only by default.
  • Per-user separation. Each account has its own data namespace.
  • Images: stored as Data URLs in your local/remote database.
  • No tracking: this app does not include analytics by default.

Security

Authentication uses Firebase Email/Password. Protect your Firebase keys and database rules. Recommended rule:

{
  "rules": {
    "tenants": {
      "$uid": {
        ".read": "$uid === auth.uid",
        "kv": { "$key": { ".write": "$uid === auth.uid" } }
      }
    }
  }
}
