Privacy Policy

How Inventory stores data

  • Local first: your data is stored in your browser’s LocalStorage for speed and offline use.
  • Optional Cloud Sync: when enabled in Settings, your data is mirrored to your own Firebase Realtime Database under tenants/<uid>/kv.
  • Per-user isolation: each signed-in user has their own local namespace and cloud path.

What we don’t collect

We don’t run trackers or analytics. Any YouTube embeds load from the official Iframe API (nocookie domain).

Security

  • Use strong passwords. Limit admin roles.
  • Keep Firebase rules locked to authenticated users only.
