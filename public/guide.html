User Guide

Getting started

  • Sign in or create an account.
  • Go to Inventory / Products to add items.
  • Use Tasks for team work; drag cards between lanes.
  • Export CSV anytime for reporting.

Images

Upload images with the file picker; we store them as Data URLs — no external hosting needed.

Hover thumbnails to see enlarged previews (mobile shows a preview modal on tap).

Cloud Sync

In Settings → Cloud Sync, turn it on to mirror your data to Firebase under your account.

Search

Type in the sidebar search to find posts, items, products, users, and pages. It matches partial words (e.g. “tu” will find “tuna”).
