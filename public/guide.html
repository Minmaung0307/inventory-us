User Guide

Getting started

  • Sign in (or use demo: admin@inventory.com/admin123 for local).
  • Use the sidebar to navigate: Inventory, Products, COGS, Tasks, Settings.
  • Search anything from the global search bar.

Inventory

  • Add items (name, code, type, price, stock, threshold, optional image).
  • Low/critical stock outlined automatically.
  • Export to CSV (no image column).

Products

  • Create products with barcode, price, ingredients and instructions.
  • Click thumbnail to view a product card modal.

COGS

  • Add daily rows; totals and gross profit calculate automatically.
  • Export to CSV for accounting.

Tasks

  • Drag tasks between lanes (To-do / In progress / Done).
  • On mobile, tap a task to cycle its lane.

Settings & Users

  • Enable/disable Cloud Sync (per-user Firebase path).
  • Manage users with role-based permissions.
  • Change theme and font size.
