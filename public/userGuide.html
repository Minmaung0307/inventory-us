Inventory & Products App — User Guide

English + Myanmar (မြန်မာ)

Quick jumps

Goal: help your team add items/products, upload optional images (JPG/PNG), search quickly, and export CSV. This guide matches the features present in your current build.

ရည်ရွယ်ချက်: သင့်အသင်းက အရင်းအစာများ/ကုန်ပစ္စည်းများကို ထည့်သွင်း ပြင်ဆင်ရန်၊ ရုပ်ပုံကို (JPG/PNG) ထည့်ရန်၊ အမြန်ရှာရန်၊ CSV ထုတ်ယူရန် လွယ်ကူစေဖို့ ဒီလမ်းညွှန်ရေးထားတာ ဖြစ်ပါတယ်။

1) Login & Sign up

  1. Enter your Email and Password, then press Sign in.
  2. New user? Click Sign up to create an account (default password allowed by the app).
  3. Forgot password? Use Forgot? to receive a reset email.

အသစ်ဝင်ရောက်ရန် Email + Password ဖြည့်ပြီး Sign in နှိပ်ပါ။ အကောင့်မရှိသေးရင် Sign up နှိပ်ပြီး အကောင့်ဖန်တီးနိုင်ပါတယ်။ Password မေ့နေပါက Forgot? ကိုနှိပ်ပြီး Reset Email ရယူနိုင်ပါတယ်။

Tech: Firebase Auth (email/password)

Under the hood

  • Sign in via auth.signInWithEmailAndPassword.
  • Sign up creates a registry doc and sets role (admin if email is whitelisted).
  • Password reset uses auth.sendPasswordResetEmail.

အထဲမှာ အကောင့်ဝင်ခြင်း၊ အကောင့်ဖန်တီးခြင်း၊ Password Reset စသည်များကို Firebase Auth နဲ့ ဆောင်ရွက်ပါတယ်။

2) Navigation

Use the sidebar to switch between: Dashboard, Inventory, Products, COGS, Tasks, Settings, Links, and Search.

ဘေးဘားမှ Dashboard, Inventory, Products, COGS, Tasks, Settings, Links, Search တို့ကိုရွေးချယ်ပို့နိုင်ပါတယ်။

3) Inventory

Add an item

  1. Click Add Item.
  2. Fill Name, Code, Type, Price, Stock, Threshold.
  3. Press Save. (Name is required.)

Add Item နှိပ်ပြီး အချက်အလက်များ ဖြည့်ပါ (Name, Code, Type, Price, Stock, Threshold)။ Save နှိပ်ပါ (Name မဖြည့်လျှင် မသိမ်းပါ）。

Optional: attach an image

If your “Add Item” modal includes Image URL and/or File, you can attach a JPG/PNG. Files are uploaded to Firebase Storage and the public URL is saved to imageUrl. URL input is also accepted.

  • Accepted file types: image/jpeg, image/png.
  • Stored path format: tenants/<uid>/inventory/<docId>/image.<ext>.

မော်ဒယ်ထဲမှာ Image URL / File ရှိပါက JPG/PNG ကို ထည့်နိုင်ပါတယ်။ File ကို Firebase Storage သို့တင်ပြီး imageUrl အနေဖြင့် သိမ်းမည်။ URL ထည့်ခြင်းလည်း လက်ခံပါတယ်။

Edit, delete, and quick stock +/-

  • Edit or Delete via row buttons (permission required).
  • Use +/- buttons on a row to adjust stock by one.

တန်းစီမှုအတန်းတွင် Edit/Delete ခလုတ်များ (ခွင့်ပြုချက်လို)၊ +/- ဖြင့် stock ကို တစ်လှမ်းစီ ပြောင်းလဲနိုင်ပါတယ်။

4) Products

Add a product

  1. Click Add Product.
  2. Fill basic fields (Name, Barcode, Price, Type, etc.).
  3. Optionally add Image URL or select a File (JPG/PNG). On save, the file is uploaded and imageUrl is stored.

Add Product နှိပ်ပြီး အချက်အလက်များ ဖြည့်ပါ။ Image URL သို့မဟုတ် File (JPG/PNG) ထည့်နိုင်ပြီး သိမ်းချိန်တွင် ကိုယ်တိုင် upload လုပ်ပြီး imageUrl သိမ်းမည်။

5) Search

Use the top search or sidebar search: type to see live suggestions, press Enter to jump, or click a suggestion result to navigate directly.

အပေါ်ဘား သို့မဟုတ် ဘေးဘားရှာဖွေမှုတွင် စာရိုက်လိုက်တာနဲ့ အကြံပြုစာရင်းတွေ မြင်ရပြီး Enter နှိပ်ပြီး သွားနိုင်သလို ရလဒ်တန်းကို နှိပ်ပြီး လှည့်ပတ်နိုင်ပါတယ်။

6) Export

Inventory → click Export CSV to download a spreadsheet of current rows.

Inventory မှာ Export CSV ကို နှိပ်ပြီး လက်ရှိအချက်အလက်များကို spreadsheet အဖြစ် ဒေါင်းလုဒ်နိုင်ပါတယ်။

7) Roles & Permissions

The app enforces permissions. Typical actions require “can add/edit/delete” depending on your role (e.g., admin or manager). If you see “No permission”, contact an admin to adjust your role.

ခွင့်ပြုချက်စနစ် ရှိပါတယ်။ မည်သည့်လုပ်ဆောင်ချက်တွင်မဆို သင့်အခန်းကဏ္ဍအလိုက် add/edit/delete ခွင့်လိုအပ်နိုင်ပါတယ်။ “No permission” ပြပါက admin ကို ဆက်သွယ်ကာ အခန်းကဏ္ဍ ပြင်ဆင်ပေးရန် တောင်းဆိုပါ။

8) Troubleshooting

  • “Name required” → Fill the Name field before saving (Inventory & Products).
  • “Login failed” → Check email and password; try Forgot? to reset.
  • “No permission” → Your role lacks access for that action. Ask an admin.
  • Image upload → Only JPG/PNG; use URL if you already host the image.

“Name required” → Name ကို ဖြည့်ပါ။ “Login failed” → Email/Password စစ်ပါ၊ Reset လုပ်ပါ။ “No permission” → ခွင့်မရှိသေးပါ။ Admin ကို ဆက်သွယ်ပါ။ Image upload → JPG/PNG သာ လက်ခံပါသည်။ URL ရှိရင်လည်း ထည့်နိုင်ပါသည်။
